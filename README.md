# Hello World - Gym Website

Welcome to our Hello World gym website! This is a modern, responsive fitness website built with React and TypeScript.

## Getting Started

This project is your "Hello World" introduction to building a professional gym website with modern web technologies.

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`

## What's Inside

This hello world gym website includes:

- **Home Section** - Welcome visitors with an engaging hero section
- **Benefits** - Showcase the advantages of joining your gym  
- **Our Classes** - Display available fitness classes
- **Contact Us** - Contact form and information
- **Responsive Design** - Works perfectly on desktop and mobile

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run code linting

## Tech Stack

- **React** - Frontend framework
- **TypeScript** - Type-safe JavaScript
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Smooth animations
- **React Hook Form** - Form handling
